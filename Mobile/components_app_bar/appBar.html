Method used to create and display the app bar for some screen needed.

context the context of the screen where we want to put the app bar. appBarTitle a string for the title of the screen how needs to be setted in the translations files. helpDialogContent a string for the help content of the dialog box how needs to be setted in the translations files.

Implementation

AppBar appBar(context, String appBarTitle, String helpDialogContent){
  return AppBar(
      leading: IconButton(
        key: const Key("backButton"),
        icon: const Icon(Icons.arrow_back, size: 32),
        onPressed: () => Navigator.of(context).pop(),
      ),
      title: Text(appBarTitle, style: const TextStyle(fontSize: 28),).tr(),
      actions: [
        helpDialogContent.isNotEmpty ? IconButton(
          onPressed: () => showDialog(
            context: context,
            builder: (context) => HelpDialog(
              title: 'help'.tr(), content: helpDialogContent.tr())),
          icon: const Icon(Icons.info_outline_rounded),
          iconSize: 32,
        ) : const SizedBox(),
      ],
    );
}
